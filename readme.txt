WP eSmiler is an emoticon plug-in for WordPress platform.

== Description ==

WP eSmiler is an emoticon plug-in for WordPress platform, created by Phong Thai, from <a href="http://www.javascriptbank.com/">JavaScriptBank.com</a>. This smiley plug-in is super easy to use and setup, update new emoticon sets by one click.

== Installation ==

e.g.

1. Unzip then upload "wp-esmiler" folder to the "/wp-content/plugins/" directory
2. Activate the plugin through the 'Plugins' menu in WordPress
